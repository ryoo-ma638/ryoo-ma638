### つくる人 — 大崎稜馬

愛知工業大学 情報科学部でメディア情報を専攻し、水野慎士研究室で画像処理とコンピュータグラフィックスを学んでいます。

ゲーム・アプリ・Web と、つくるものの形にはこだわらず、「人が触って面白いもの」を企画から実装まで手を動かしてつくるのが好きです。創作系サークル CEED の副部長もしていて、野球場ではビールの売り子もしています。

### これまでにつくったもの

- **虫歯アニマルズ** — 圧力センサー付き歯ブラシで動物の歯を磨く体感ゲーム。東京ゲームショウ2024・ゲームマーケット2024秋に出展し、日経クロストレンドにも取り上げられました。企画と3Dモデリングを担当。
- **[Settlo](https://github.com/ryoo-ma638/Settlo)** — みんなで使える割り勘アプリ。サークル内ハッカソン「シスハック2026」で企業協賛賞を受賞。5人チームで開発。
- **[meegri](https://meegri.vercel.app/)** — オンラインの個別通話を本番そっくりに練習できる Web アプリ。生成AI（Gemini API）と音声合成で、キャラクターと自然に会話できます。企画から実装まで一人で。
- **ドローン点検アプリ** — ある企業から受託した、ドローンの飛行記録を扱う業務アプリ。企画から納品まで担当（受託案件のため詳細は非公開）。
- **LiDAR鎌ゲーム** — LiDAR で足の動きを追跡し、鎌型のコントローラを振って遊ぶ設置型アクション。
- **スマート傘（研究）** — カメラで雨の降る向きを直接計測する傘。WISS 2026 での発表を目標に研究中。
